## **6.3 [実験](https://ja.wikipedia.org/wiki/%E5%AE%9F%E9%A8%93)系の設計**
### **6.3.1 [系](https://ja.wikipedia.org/wiki/%E7%B3%BB)・[因子](https://ja.wikipedia.org/wiki/%E5%9B%A0%E5%AD%90)・[結果](https://ja.wikipedia.org/wiki/%E7%B5%90%E6%9E%9C)**
- 因子（群）・結果（群）
	- 物質：部品、補助因子、など
	- 条件：場所、時間・タイミング、温度・湿度・場など
	- [事象](https://ja.wikipedia.org/wiki/%E4%BA%8B%E8%B1%A1)・関係性：運動の変化、形体の変化、反応、結合、分解、意味合いの変化、など
	- 因子水準、数量化された観測変数
	- [潜在変数](https://ja.wikipedia.org/wiki/%E6%BD%9C%E5%9C%A8%E5%A4%89%E6%95%B0)
- 因子の特徴
	- 機能：他の因子への作用、機能が現れる環境やタイミング
	- 性質：形状、量など、変化・他の因子からの影響
	- 起源・将来：出所、進化、分化、統合など
- 系のパターンとルール
	- 因子間の関係性
	- 系のルール・パターン・目的
	- ルールが守られなかった時の対応
- 応用可能性
    - 因子の改変、条件変更、新たな組み合わせ
    - ルールの応用、新たなルール
    - スケールアップ・ダウン、等

### **6.3.6 結果（群）Bに対する因子（群）Aの同定（参考）**
- Aは本当に因子か
	- 再現性の確認は取れるか、データの選択バイアスによる過学習モデルとなっていないか
		- 繰り返し再現性、別の測定法による再現性、クロスバリデーションは可能か
		- 他の研究者による再現性取得は可能か
	- Bが生じて（存在して）いる時には必ずAが存在しているか
		- Aを無くすことで常にBが生じないか
		- Aが無い状態からAを追加することでBが生じるか
	- BとAは量に相関があるか
	- AとBの共通の要因となっている因子（群）Cは存在しないか、潜在因子、[交絡](https://ja.wikipedia.org/wiki/%E4%BA%A4%E7%B5%A1)
	- BがAの因子である可能性は無いか
		- 相互作用、シナジー
	- AとBが一致している（同じ現象を他の見方をしている）可能性は無いか
- Aは直接的要因か
	- Aは因子（群）Cに働き、CがBの直接的要因である可能性は無いか
		- AとBは直接的な関係があるか（結合・情報伝達）
	- 補助因子
	- 促進因子・抑制因子
	- AはB以外にも影響を与えていないか（B以外の観測変数の有無、系そのものを壊していないか）
- Aの分解可能性は無いか：主要因、特徴量抽出
	- Aが存在していてもBが存在しない場合はあるか、それはどのような場合か
	- 因子水準を変更した場合にBが生じるAの範囲・条件はあるか
	- Aを類似のものに変更・一部改変、系のスケールアップ・ダウン、などできるか
	- 因子群の関係性は把握可能か
		- 交差負荷
- 因子はAだけか
	- AだけでBが生じることを別の系（場所・時間等）で確認できるか
		- 再構成
	- A以外に因子となる可能性があるものは無いか、A以外のものでBを再現できないか
		- [リバースエンジニアリング](https://ja.wikipedia.org/wiki/%E3%83%AA%E3%83%90%E3%83%BC%E3%82%B9%E3%82%A8%E3%83%B3%E3%82%B8%E3%83%8B%E3%82%A2%E3%83%AA%E3%83%B3%E3%82%B0)
	- A -> B, B -> Cが示せても、A,B,Cが同時に存在する時にはCがA -> Bであったり、AがB -> Cに干渉し、A -> B -> Cとならない可能性は無いか
