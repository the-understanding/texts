6.3 実験系の設計

6.3.1 系・因子・結果

  • 因子（群）・結果（群）
    • 物質：部品、補助因子、など
    • 条件：場所、時間・タイミング、温度・湿度・場など
    • 事象・関係性：運動の変化、形体の変化、反応、結合、分解、意味合いの変化、など
    • 因子水準、数量化された観測変数
    • 潜在変数
  • 因子の特徴
    • 機能：他の因子への作用、機能が現れる環境やタイミング
    • 性質：形状、量など、変化・他の因子からの影響
    • 起源・将来：出所、進化、分化、統合など
  • 系のパターンとルール
    • 因子間の関係性
    • 系のルール・パターン・目的
    • ルールが守られなかった時の対応
  • 応用可能性
    • 因子の改変、条件変更、新たな組み合わせ
    • ルールの応用、新たなルール
    • スケールアップ・ダウン、等

6.3.2 定性・定量

  • 定性分析
  • 定量分析
  • 尺度水準
  • データの規格化

6.3.3 量

  • 国際単位系における７つの基本量：時間、長さ、質量、電流、熱力学温度、物質量、光度 (光学)
  • 単位 、法定計量単位、計量法に基づく計量単位一覧

6.3.4 探索空間・データサンプリングの設計

  • 予備検討、予備実験、予備調査
  • 背景情報・モデルに用いるデータの収集、選別、データの収集方法の決定
  • 探索空間の設計
    • クオリティ・バイ・デザイン
    • 実験計画法
      • 応答曲面法
      • サロゲートモデル
  • 臨床研究
    • 観察研究
    • 介入研究
  • メタアナリシス
  • バイアスの低減（-> 9.4.3 バイアス・誤謬・詭弁）
    • 無作為抽出、層化抽出法、乱塊法
      • Randomized experiment
        • ランダム化比較試験
    • 自然実験
    • 二重盲検法
    • 偽薬効果
  • サンプル数、標本 (統計学)
  • サンプルサイズ
    • 検出力、効果量（Effect size）
  • 連続実施と間隔を開けた実施

6.3.5 因子（群）Aと結果（群）Bの関係

  • 関連性
  • パス図
  • 相関、偏相関、擬似相関
  • 因果性（-> 4.2 因果性）
  • 交絡

6.3.6 結果（群）Bに対する因子（群）Aの同定（参考）

  • Aは本当に因子か
    • 再現性の確認は取れるか、データの選択バイアスによる過学習モデルとなっていないか
      • 繰り返し再現性、別の測定法による再現性、クロスバリデーションは可能か
      • 他の研究者による再現性取得は可能か
    • Bが生じて（存在して）いる時には必ずAが存在しているか
      • Aを無くすことで常にBが生じないか
      • Aが無い状態からAを追加することでBが生じるか
    • BとAは量に相関があるか
    • AとBの共通の要因となっている因子（群）Cは存在しないか、潜在因子、交絡
    • BがAの因子である可能性は無いか
      • 相互作用、シナジー
    • AとBが一致している（同じ現象を他の見方をしている）可能性は無いか
  • Aは直接的要因か
    • Aは因子（群）Cに働き、CがBの直接的要因である可能性は無いか
      • AとBは直接的な関係があるか（結合・情報伝達）
    • 補助因子
    • 促進因子・抑制因子
    • AはB以外にも影響を与えていないか（B以外の観測変数の有無、系そのものを壊していないか）
  • Aの分解可能性は無いか：主要因、特徴量抽出
    • Aが存在していてもBが存在しない場合はあるか、それはどのような場合か
    • 因子水準を変更した場合にBが生じるAの範囲・条件はあるか
    • Aを類似のものに変更・一部改変、系のスケールアップ・ダウン、などできるか
    • 因子群の関係性は把握可能か
      • 交差負荷
  • 因子はAだけか
    • AだけでBが生じることを別の系（場所・時間等）で確認できるか
      • 再構成
    • A以外に因子となる可能性があるものは無いか、A以外のものでBを再現できないか
      • リバースエンジニアリング
    • A -> B, B -> Cが示せても、A,B,Cが同時に存在する時にはCがA -> Bであったり、AがB -> Cに干渉し、A -> B -> Cとならない可能性は無いか
